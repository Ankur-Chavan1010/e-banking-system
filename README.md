# E-Banking System

Enterprise-level banking application built using:

- Spring Boot
- Spring Security
- JWT
- MySQL
- React
- Microservices Architecture

## Features

- Authentication & Authorization
- Account Management
- Transactions
- Payment Services
- Email Notifications
- Loan Management

## Implemented Features

- JWT Authentication
- Refresh Token Mechanism
- Spring Security
- Role-Based Authorization
- Protected REST APIs
- Global Exception Handling
- Validation
- UUID-Based Database Design
